# Πελάτες

Η ενότητα **Πελάτες** σας επιτρέπει να διαχειρίζεστε όλους τους επιβάτες ή πελάτες που χρησιμοποιούν τις μεταφορές και εκδρομές σας. Εδώ μπορείτε να δημιουργείτε, να βλέπετε και να επεξεργάζεστε πληροφορίες πελατών αποτελεσματικά.

---

## Δημιουργία Πελάτη

Μπορείτε να δημιουργήσετε χειροκίνητα έναν νέο πελάτη παρέχοντας τις εξής λεπτομέρειες:

- **Όνομα (Name)** – Πλήρες όνομα πελάτη  
- **Email** – Διεύθυνση email για ειδοποιήσεις και επικοινωνία  
- **Χώρα (Country)** – Χώρα διαμονής του πελάτη  
- **Διεύθυνση (Address)** – Οδός / Διεύθυνση  
- **Πόλη (City)** – Πόλη διαμονής  
- **Τ.Κ. (Postcode)** – Ταχυδρομικός Κωδικός  
- **Τηλέφωνο (Phone)** – Αριθμός επικοινωνίας  
- **Σημειώσεις (Notes)** – Οποιαδήποτε πρόσθετη πληροφορία θέλετε να αποθηκεύσετε  

Αφού συμπληρώσετε τα πεδία, αποθηκεύστε τον πελάτη για να είναι διαθέσιμος για μελλοντικές κρατήσεις.

---

## Αυτόματη Δημιουργία Πελατών

Κάθε φορά που γίνεται μια κράτηση στο Transfervista, δημιουργείται αυτόματα ένα νέο αρχείο πελάτη αν ο πελάτης δεν υπάρχει ήδη.  

Αυτό διασφαλίζει ότι κάθε επιβάτης ή πελάτης παρακολουθείται στο σύστημα χωρίς να χρειάζεται χειροκίνητη εισαγωγή κάθε φορά.

---

## Διαχείριση Πελατών

Αφού δημιουργηθεί, μπορείτε να:

- Δείτε όλα τα προφίλ πελατών  
- Επεξεργαστείτε στοιχεία όπως πληροφορίες επικοινωνίας, σημειώσεις και διεύθυνση  
- Αναζητήσετε και φιλτράρετε πελάτες για να βρείτε γρήγορα τον σωστό  

Η σωστή διαχείριση πελατών διευκολύνει τις κρατήσεις, την επικοινωνία και την έκδοση αναφορών.
