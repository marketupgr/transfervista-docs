# Πώς να προσφέρετε μεταφορές μόνο μεταξύ προκαθορισμένων σημείων παραλαβής & παράδοσης

Αυτός ο οδηγός εξηγεί πώς να ρυθμίσετε το TransferVista ώστε οι πελάτες να μπορούν **να κάνουν κράτηση μόνο μεταξύ συγκεκριμένων προκαθορισμένων σημείων**, όπως:

- Αεροδρόμιο → Ξενοδοχείο  
- Ξενοδοχείο → Αεροδρόμιο  
- Οποιοδήποτε άλλο ζευγάρι προκαθορισμένων τοποθεσιών  

Αυτή η ρύθμιση είναι ιδανική για εταιρείες που **δεν** προσφέρουν ελεύθερες διευθύνσεις και θέλουν να περιορίσουν τις κρατήσεις μόνο σε συγκεκριμένες διαδρομές.

---

## 1. Προσθέστε τα Οχήματά σας  
Πριν ορίσετε τιμές ή διαδρομές, πρέπει να δημιουργήσετε τα οχήματά σας.  
Όλοι οι κανόνες τιμολόγησης πρέπει να συνδέονται με κάποιο όχημα.

Βήματα:

1. Μεταβείτε στο **Ρυθμίσεις → Οχήματα**
2. Πατήστε **Προσθήκη Οχήματος**
3. Συμπληρώστε τα στοιχεία (χωρητικότητα, αποσκευές, τύπος οχήματος κ.λπ.)
4. Αποθηκεύστε το όχημα
5. Επαναλάβετε για όσα οχήματα χρειάζεστε

> **Σημαντικό:** Δεν μπορείτε να δημιουργήσετε κανόνες τιμολόγησης χωρίς να υπάρχουν οχήματα.

---

## 2. Δημιουργήστε Ζώνες Διαθεσιμότητας  
Οι Ζώνες Διαθεσιμότητας καθορίζουν πού μπορεί να λειτουργεί η εταιρεία σας.  
Οι πελάτες *δεν* μπορούν να κάνουν κράτηση εκτός των ζωνών αυτών.

Βήματα:

1. Μεταβείτε στο **Ρυθμίσεις → Ζώνες Διαθεσιμότητας**
2. Πατήστε **Δημιουργία Ζώνης**
3. Σχεδιάστε μια ζώνη γύρω από την πρώτη σας τοποθεσία (π.χ. αεροδρόμιο)
4. Δημιουργήστε δεύτερη ζώνη γύρω από τη δεύτερη τοποθεσία (π.χ. ξενοδοχείο)
5. Αποθηκεύστε και τις δύο ζώνες

**Παράδειγμα:**

- Ζώνη 1 → Αεροδρόμιο Ελευθέριος Βενιζέλος  
- Ζώνη 2 → Hilton Hotel  

Με αυτές τις δύο ζώνες, το σύστημα επιτρέπει κρατήσεις **μόνο μεταξύ αυτών των δύο σημείων**.

---

## 3. Δημιουργήστε Περιοχές για Τα Σταθερά Σημεία  
Οι Περιοχές χρησιμοποιούνται για να ορίσουν **σταθερά σημεία παραλαβής/παράδοσης** που εμφανίζονται:

- Στο πεδίο διεύθυνσης της φόρμας κράτησης  
- Στη λίστα τιμών (όταν τιμολογείτε ανά διαδρομή)

Βήματα:

1. Μεταβείτε στο **Ρυθμίσεις → Περιοχές**
2. Δημιουργήστε την πρώτη Περιοχή (π.χ. Αεροδρόμιο Ελευθέριος Βενιζέλος)
3. Σχεδιάστε το σημείο/όριο στον χάρτη
4. Ενεργοποιήστε το **Εμφάνιση αυτής της περιοχής στη φόρμα κράτησης**
5. Επαναλάβετε για τη δεύτερη τοποθεσία (π.χ. Hilton Hotel)
6. Αποθηκεύστε

---

## 4. Δημιουργήστε Κανόνες Τιμολόγησης (Ανά Διαδρομή)

Με τα οχήματα, τις ζώνες και τις περιοχές έτοιμες, μπορείτε να ορίσετε τις σταθερές τιμές σας:

1. Μεταβείτε στο **Ρυθμίσεις → Λίστα Τιμών**
2. Πατήστε **Νέες Καταχωρήσεις**
3. Επιλέξτε **Προσθήκη Τιμών ανά Διαδρομή**
4. Διαλέξτε το όχημα που θα χρησιμοποιηθεί
5. Επιλέξτε:
   - **Περιοχή Παραλαβής**
   - **Περιοχή Παράδοσης**
6. Ορίστε τη σταθερή τιμή της διαδρομής
7. Ενεργοποιήστε το **Αντίστροφη Διαδρομή** για να δημιουργηθεί αυτόματα και η αντίστροφη τιμή  
8. Αποθηκεύστε

Επαναλάβετε για όλα τα οχήματα ή επιπλέον διαδρομές.

---

## 5. Δοκιμάστε τη Φόρμα Κράτησης  
Ελέγξτε ότι όλα λειτουργούν σωστά:

1. Ανοίξτε τη δημόσια φόρμα κράτησης  
2. Επιλέξτε σημείο παραλαβής  
3. Βεβαιωθείτε ότι εμφανίζονται μόνο οι προκαθορισμένες περιοχές  
4. Επιλέξτε σημείο παράδοσης  
5. Ελέγξτε την εμφανιζόμενη τιμή  
6. Επιβεβαιώστε ότι **δεν υπάρχουν άλλες διαδρομές ή επιλογές ελεύθερης διεύθυνσης**

---

## ✅ Σύνοψη

Για να επιτρέψετε κρατήσεις **μόνο μεταξύ προκαθορισμένων διαδρομών**, πρέπει:

1. **Να δημιουργήσετε Οχήματα**  
2. **Να σχεδιάσετε Ζώνες Διαθεσιμότητας**  
3. **Να δημιουργήσετε Περιοχές**  
4. **Να προσθέσετε Κανόνες Τιμολόγησης ανά Διαδρομή**  
5. **Να δοκιμάσετε τη Φόρμα Κράτησης**

Με αυτές τις ρυθμίσεις, οι πελάτες θα μπορούν να κάνουν κράτηση **μόνο για τις διαδρομές που έχετε ορίσει**, με σταθερές τιμές και χωρίς ελεύθερες διευθύνσεις.
