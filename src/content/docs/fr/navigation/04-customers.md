# Clients

La section **Clients** vous permet de gérer tous les passagers ou clients qui utilisent vos transferts et excursions.  
Vous pouvez créer, consulter et modifier les informations clients de manière efficace.

---

## Création d’un client

Vous pouvez créer manuellement un nouveau client en renseignant les informations suivantes :

- **Nom** – Nom complet du client  
- **Email** – Adresse e-mail pour les notifications et la communication  
- **Pays** – Pays de résidence du client  
- **Adresse** – Adresse postale  
- **Ville** – Ville de résidence  
- **Code postal** – Code postal  
- **Téléphone** – Numéro de contact  
- **Notes** – Toute information supplémentaire que vous souhaitez conserver  

Après avoir rempli ces champs, enregistrez le client pour qu’il soit disponible pour les futures réservations.

---

## Création automatique de clients

Chaque fois qu’une réservation est effectuée dans Transfervista, un nouveau client est automatiquement créé si celui-ci n’existe pas déjà.  

Cela garantit que chaque passager ou client est suivi dans le système sans nécessiter une saisie manuelle à chaque fois.

---

## Gestion des clients

Une fois créés, vous pouvez :

- Consulter tous les profils clients  
- Modifier les informations telles que les coordonnées, les notes et l’adresse  
- Rechercher et filtrer les clients pour trouver rapidement celui que vous cherchez  

Une bonne gestion des clients permet de rationaliser les réservations, la communication et le reporting.
