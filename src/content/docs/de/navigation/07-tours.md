# Touren

### Erstellen Ihrer ersten Tour

![new tour](../../../../assets/tours.png)

1. **Zugriff auf das Touren-Modul**
   - Melden Sie sich im TransferVista-Admin-Panel an
   - Navigieren Sie zu `Einstellungen > Touren` in der linken Seitenleiste
   - Klicken Sie oben rechts auf die blaue Schaltfläche `Neue Tour`

2. **Grundinformationen festlegen**
   
   Füllen Sie die wichtigsten Tourdetails aus:
   
   - **Titel**: Geben Sie der Tour einen klaren, beschreibenden Namen (z. B. „Lasithi-Plateau & Kretisches Wein- und Olivenöl-Erlebnis“)  
   - **Kurzbeschreibung**: Schreiben Sie eine 1–2 Sätze lange Zusammenfassung für die Tourübersicht  
   - **Kategorie**: Weisen Sie eine standortbasierte Kategorie zu (z. B. Lasithi, Samaria, Chania)  
   - **Aktivitätstyp**: Geben Sie den Tourtyp an (Wein-Erlebnis, Schlucht, Abenteuer, Archäologisch, Besuch)

![new tour details](../../../../assets/tour-details.png)

3. **Detaillierte Beschreibung erstellen**
   
   Verwenden Sie den Rich-Text-Editor für die vollständige Tourbeschreibung:
   
   - Schreiben Sie einen einleitenden, ansprechenden Absatz  
   - Fügen Sie eine **Highlights**-Sektion mit Aufzählungspunkten für wichtige Sehenswürdigkeiten hinzu  
   - Beschreiben Sie einzelne Stationen oder Aktivitäten detailliert  
   - Formatieren Sie mit Fett, Kursivschrift und Listen für bessere Lesbarkeit


4. **Tourdetails konfigurieren**
   
   Legen Sie die Betriebsparameter fest:
   
   - **Ort**: Hauptstandort  
   - **Abfahrtsort**: Ort der Abholung der Kunden (z. B. „Heraklion“)  
   - **Abfahrtszeit**: Startzeit im 24-Stunden-Format (z. B. 12:00)  
   - **Dauer**: Länge in Stunden  
   - **Kapazität**: Maximale Teilnehmerzahl (angezeigt als „Bis zu X Erwachsene“)

5. **Bilder hinzufügen**
   
   - Bis zu 6 hochwertige Bilder hochladen (max. 8 MB pro Datei)  
   - Das erste Bild wird zum Hauptbild der Tour  
   - Verwenden Sie Landschaftsbilder, die die wichtigsten Attraktionen zeigen  
   - Klicken Sie auf `Entfernen`, um unerwünschte Bilder zu löschen

6. **Preisgestaltung festlegen**
   
   Konfigurieren Sie die Preisstruktur:
   
   - Preis für Erwachsene (angezeigt als „ab €XX,XX pro Erwachsener“)  
   - Preis für Kinder (angezeigt als „ab €XX,XX pro Kind“)  
   - Preise werden mit Basispreis prominent angezeigt

7. **Tour speichern**
   
   Klicken Sie oben rechts auf die blaue Schaltfläche `Speichern`

## Einbindung der Touren auf Ihrer Website

TransferVista stellt fertige HTML-Embed-Codes bereit, um Ihre Touren professionell auf jeder Website anzuzeigen.

### Verfügbare Embed-Optionen

#### 1. Tourenübersicht (Komplettkatalog)

Zeigt alle Touren in einem filterbaren Grid-Layout mit Sortier- und Filterfunktionen.

**Funktionen:**
- Sortierung (Standard, Preis, Dauer etc.)  
- Standortfilter-Dropdown  
- Filter nach Aktivitätstyp  
- Kategoriefilter  
- Responsives Kartenlayout mit Bildern  
- Preisanzeige  
- „Jetzt buchen“-Buttons

#### 2. Touren-Slider (Karussell)

Zeigt Touren in einem interaktiven Karussell, ideal für Startseiten oder Werbebereiche.

### So binden Sie Touren auf Ihrer Website ein

1. **Embed-Codes abrufen**
   - Melden Sie sich im Admin-Panel an  
   - Navigieren Sie zu `Einstellungen > Formulare & Dokumente`  
   - Klicken Sie auf den Reiter `Touren`  
   - Sie sehen zwei Bereiche: „Tourenübersicht“ und „Touren-Slider“

![embed tour](../../../../assets/embed-form-for-tour-booking.png)

2. **Anzeigestil auswählen**
   - **Tourenübersicht**: Zeigt alle Touren in einem Grid mit Filtern  
   - **Touren-Slider**: Zeigt Touren in einem Karussell

3. **Embed-Code kopieren**
   - Jede Sektion enthält einen Code-Block mit HTML  
   - Klicken Sie in den Code-Block, um alles auszuwählen  
   - Kopieren Sie den gesamten Code (in der Regel 2 Zeilen)  

4. **Auf der Website einfügen**
   - Öffnen Sie den Website-Editor (WordPress, Wix, Squarespace, HTML etc.)  
   - Gehen Sie zu der Seite, auf der die Touren angezeigt werden sollen  
   - Fügen Sie den Code in den HTML-Bereich ein  
   - Bei Page-Buildern: „HTML“ oder „Custom Code“-Block verwenden  
   - Der Code funktioniert auf allen Plattformen, keine weitere Konfiguration nötig

![tour carousel](../../../../assets/carousel-of-tours-booking-form.png)

5. **Integration testen**
   - Seite speichern und veröffentlichen  
   - Website besuchen, um Touren anzuzeigen  
   - Buchungsablauf testen („Jetzt buchen“)  
   - Filter und Sortierung prüfen  
   - Auf mobilen Geräten responsive Anzeige prüfen

![tour booking](../../../../assets/tour-booking-form.png)

## Der Buchungsprozess

### Kundenbuchung

Kunden erleben einen nahtlosen 3-Schritte-Buchungsprozess:

**Schritt 1: Tour auswählen**
- Verfügbare Touren mit Bildern und Beschreibungen durchsuchen  
- Wichtige Details sehen: Abfahrtszeit, Ort, Preis  
- Vollständige Beschreibung und Highlights lesen  
- „Jetzt buchen“ klicken

**Schritt 2: Details eingeben**
- Buchungsdatum auswählen  
- Anzahl der Erwachsenen wählen  
- Anzahl der Kinder wählen  
- Preisaktualisierung wird automatisch angezeigt  
- Hinweis: „* Datum und Preis unterliegen der Verfügbarkeit“  
- Auf „Weiter“ klicken

**Schritt 3: Zahlung abschließen**
- Kundendaten eingeben  
- Zahlung verarbeiten  
- Bestätigung erhalten

### Anzeige der Touren

Jede Tour-Karte zeigt:
- Hochwertiges Titelbild  
- Tourtitel  
- Vorschau der Kurzbeschreibung  
- Abfahrtsort mit Pin-Icon  
- Abfahrtszeit mit Uhr-Icon  
- Dauer mit Uhr-Icon  
- Kapazität  
- Preis (Erwachsene und Kinder)  
- „Jetzt buchen“-Button

## Tourbuchungen verwalten

### Buchungen einsehen
- Über die **Buchungen**-Sektion in der linken Seitenleiste  
- Alle Tourbuchungen zusammen mit Transferbuchungen anzeigen  
- Filtern und sortieren

### Tourverwaltung
- Tourenliste zeigt: Titel, Ort, Aktivitätstyp, Kategorie, Optionen (⋮)  
- Schnellaktionen: Tour bearbeiten, Buchungen einsehen, duplizieren, archivieren/löschen

## Häufige Fragen

### Einrichtung & Konfiguration
- **Wie viele Touren kann ich erstellen?** Keine Begrenzung  
- **Kann ich das Erscheinungsbild der eingebetteten Touren anpassen?** Embed-Codes sind professionell und responsiv  
- **Unterstützte Bildformate?** JPG, PNG bis 8MB, Landschaftsbilder empfohlen  
- **Unterschiedliche Preise für verschiedene Daten?** Basispreis über Preisliste, saisonale Preise prüfen  
- **Buchungen in mehreren Sprachen?** Sprache über „EN“-Selector einstellen

### Buchungsverwaltung
- **Zahlung durch Kunden?** Schritt 3 im Buchungsprozess  
- **Kapazitätsgrenzen?** Im Touren-Setup einstellbar  
- **Ausgebuchte Touren?** System verwaltet automatisch die Verfügbarkeit  
- **Buchungen am selben Tag möglich?** Ja, sofern Kapazität vorhanden

### Website-Integration
- **Codierungskenntnisse nötig?** Nein, HTML-Code kopieren und einfügen  
- **Funktioniert auf Mobilgeräten?** Ja, responsive  
- **Einbettung auf mehreren Seiten?** Ja, gleiche Codes mehrfach nutzbar  
- **Updates erscheinen sofort?** Ja, Änderungen in Admin erscheinen sofort

### Erweiterte Funktionen
- **Geteilte oder private Touren?** Unterschiedliche Kapazitäten oder separate Listings  
- **Sonderwünsche der Kunden?** Kundenverwaltung und E-Mail-Vorlagen nutzen  
- **Integration mit bestehendem Buchungssystem?** Unter Einstellungen > Integrationen prüfen  
- **Berichte für Tourleistung?** Unter Berichte einsehbar

## Best Practices

### Tourbeschreibungen
- Spannend einleiten  
- Highlights in Aufzählungen darstellen  
- Praktische Infos: Mitbringen, Schwierigkeitsgrad  
- Inklusivleistungen erwähnen

### Preisstrategie
- Wettbewerbsfähig, aber nachhaltig  
- Kinderpreise berücksichtigen  
- Alle Kosten einplanen  
- Saisonale Anpassungen prüfen

### Bilder
- Professionell, gut belichtet  
- Zeigt tatsächliche Orte und Erlebnisse  
- Personen in Aktion zeigen  
- Für Web optimiert

### Kundenerlebnis
- Schnell auf Anfragen reagieren  
- Informationen aktuell halten  
- Realistische Erwartungen setzen  
- Feedback einholen
