# Clientes

La sección de **Clientes** te permite gestionar a todos los pasajeros o clientes que utilizan tus traslados y tours. Aquí puedes crear, ver y editar información de clientes de manera eficiente.

---

## Crear un Cliente

Puedes crear manualmente un nuevo cliente proporcionando los siguientes detalles:

* **Nombre** – Nombre completo del cliente
* **Correo electrónico** – Dirección de correo para notificaciones y comunicación
* **País** – País de residencia del cliente
* **Dirección** – Calle y número
* **Ciudad** – Ciudad de residencia
* **Código Postal** – Código postal
* **Teléfono** – Número de contacto
* **Notas** – Cualquier información adicional que desees almacenar

Después de completar estos campos, guarda el cliente para que esté disponible para futuras reservas.

---

## Creación Automática de Clientes

Cada vez que se realiza una reserva en Transfervista, se crea automáticamente un registro de cliente si este no existe previamente.

Esto asegura que cada pasajero o cliente esté registrado en el sistema sin requerir entrada manual en cada ocasión.

---

## Gestión de Clientes

Una vez creado, puedes:

* Ver todos los perfiles de clientes
* Editar detalles como información de contacto, notas y dirección
* Buscar y filtrar clientes para encontrar rápidamente el correcto

Una gestión adecuada de clientes ayuda a optimizar reservas, comunicación e informes.
